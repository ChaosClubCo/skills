# Market Segmentation

## Purpose
Segment markets by ICP, firmographics, and buying behavior for targeted go-to-market.

## When to Use
- Strategic planning for B2B SaaS/AIaaS business
- Executive decision-making requiring frameworks
- Cross-functional strategy alignment

## Core Framework
1. **Assess Current State** - Analyze existing approach and gaps
2. **Define Strategy** - Set objectives and success criteria  
3. **Design Approach** - Create framework and implementation plan
4. **Execute** - Roll out strategy across organization
5. **Measure & Iterate** - Track KPIs and adjust based on results

## Key Deliverables
- Strategic framework (1-2 page document)
- Implementation roadmap (Gantt chart)
- Success metrics (OKRs or KPIs)
- Stakeholder communication plan

## Dependencies
Requires intinc-business-skills for operational execution.
