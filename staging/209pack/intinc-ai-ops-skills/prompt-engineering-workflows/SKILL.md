# Prompt Engineering Workflows

## Purpose
Establish repeatable workflows for prompt quality and iteration.

## When to Use
Building production prompts, inconsistent AI outputs, need version control for prompts, A/B testing prompt variations.

## Core Workflow

1. Prompt Design (define task, constraints, output format, examples)
2. Testing (create eval set, measure accuracy/latency/cost across variations)
3. Version Control (Git-based prompt library with semantic versioning)
4. A/B Testing (deploy variants, measure real-world performance)
5. Optimization (iterate based on production metrics)

## Output Format
- Architecture diagrams (Mermaid)
- Implementation guides (Markdown)
- Code examples (Python/TypeScript)
- Cost estimates (CSV/Excel)

- Foundational skill (no IntInc skill dependencies)
