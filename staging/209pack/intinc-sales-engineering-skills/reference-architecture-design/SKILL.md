# Reference Architecture Design

## Purpose
Build library of proven AI architecture patterns that accelerate sales cycles and reduce custom design work.

## When to Use
- Creating repeatable solution templates
- Standardizing architecture across multiple deals
- Training sales engineers on common patterns

## Reference Architectures
1. **Customer Support AI** - Ticket classification + suggested responses
2. **Document Q&A** - RAG system for enterprise knowledge base
3. **Sales Intelligence** - Lead scoring + email generation
4. **Code Assistant** - Context-aware code completion + review

## Each Architecture Includes
- Component diagram (Mermaid)
- Integration points
- Cost model (at 10K, 100K, 1M users)
- Security controls
- Deployment guide

## Output
Architecture library (4-6 templates) with customization guidelines
