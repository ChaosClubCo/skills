# POC Scoping

## Purpose
Define bounded proof-of-concept projects that de-risk enterprise AI purchases.

## When to Use
- Customer says: "We need to see it work with our data first"
- Enterprise procurement requires pilot before full purchase
- Technical validation needed before contract signing

## POC Framework
**Duration:** 2-4 weeks max
**Scope:** Single use case, limited user group
**Data:** Subset of production data (100-1K records)
**Success Metrics:** Quantifiable (accuracy, time savings, cost reduction)

## Deliverables
- POC plan (1-pager with timeline, success criteria, exit criteria)
- Weekly progress reports
- Final results presentation with go/no-go recommendation
