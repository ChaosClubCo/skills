# Use Case Discovery

## Purpose
Structured framework for identifying and prioritizing AI use cases with enterprise stakeholders.

## When to Use
- Initial discovery call with new enterprise prospect
- AI strategy workshop with C-suite
- Department-level automation assessment

## Workshop Structure
1. **Pain Point Collection** (30 min) - What's broken/slow/manual?
2. **Feasibility Scoring** (20 min) - Technical viability with current AI
3. **Impact Estimation** (20 min) - Hours saved, revenue impact, cost reduction
4. **Prioritization** (15 min) - Impact vs Effort matrix

## Output
Prioritized backlog of 10-15 use cases with feasibility scores and estimated ROI
