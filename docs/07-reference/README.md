## How to Use This Section

These documents are reference materials, not tutorials. Use them to:

- Look up the exact name of a constant, function, or configuration value
- Find the current model name for a specific platform and complexity level
- Check which slug collisions exist and how they are handled
- Copy-paste pipeline commands without reading the full admin guide

For step-by-step instructions, see the [Admin Guide](../06-admin-guide/README.md).
