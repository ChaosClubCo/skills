This section walks you through everything you need to go from zero to using AI skills on your platform of choice. If you have never worked with this library before, start here.

## Recommended Reading Order

1. **Quickstart** -- get one platform working end to end.
2. **Choosing a Skill** -- learn how skills are organized so you can find what you need.
3. **Platform deployment guide** -- pick the guide for your primary platform and follow it in full.
4. **FAQ** -- answers to anything the guides did not cover.

---

## Prerequisites

- Git (to clone the repository)
- Python 3.8 or later (for pipeline scripts, if needed)
- Node.js 18+ (only if running the Copilot converter)
- Access to at least one supported AI platform
