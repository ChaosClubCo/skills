# Creative Category Guide

**40 skills** -- Skills for designing, creating, and reviewing creative deliverables. This category covers visual design, motion graphics, UX research and writing, content creation, and production workflows.

**Note on miscategorized skills:** Five skills in this category are not primarily creative in nature. `clinical-trials`, `clinical-workflows`, `lean-manufacturing`, `lending-operations`, and `recruiting-hiring` were placed here during initial categorization and retain the "creative" category assignment. Their content is domain-specific and would more naturally fit in the industry or operations categories.

## Example Workflows

### Workflow 1: Designing a New Product Interface

1. **user-research-methods** -- Conduct discovery research to understand user needs
2. **user-story-writing** -- Define user stories and acceptance criteria from research
3. **design-principles** -- Apply gestalt psychology and visual hierarchy to layouts
4. **ui-design** -- Create high-fidelity interface designs with consistent patterns
5. **accessibility-core** -- Audit designs for WCAG 2.2 AA compliance
6. **responsive-design-patterns** -- Ensure layouts work across breakpoints
7. **animation-motion-design** -- Add micro-interactions and transition animations
8. **figma-production-workflow** -- Organize files for developer handoff

### Workflow 2: Building a Brand from Scratch

1. **creative-direction** -- Establish creative strategy and brand vision
2. **brand-systems** -- Design logo, color palette, typography, and visual elements
3. **color-theory-application** -- Develop accessible, harmonious color system
4. **typography-mastery** -- Select and pair typefaces for the brand system
5. **design-movements** -- Reference historical influences for aesthetic direction
6. **deliverable-templates** -- Create branded templates for common deliverables
7. **production-graphics** -- Export brand assets in all required formats

### Workflow 3: Creating a Data-Driven Report

1. **data-visualization** -- Design chart types and visual encoding for the data
2. **infographic-design** -- Transform complex findings into visual narratives
3. **presentation-design** -- Structure the narrative arc of the report
4. **color-theory-application** -- Apply meaningful color coding to data categories
5. **typography-mastery** -- Set type hierarchy for headings, body, and annotations
6. **production-graphics** -- Export final assets for print and digital distribution
