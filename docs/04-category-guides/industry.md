## Example Workflows

### Workflow 1: Setting Up a Healthcare SaaS Platform

1. **healthcare-compliance** -- Map HIPAA and HITECH requirements for the application
2. **fintech-operations** -- If handling payments, design compliant payment processing (combine with strategy/security-compliance for SOC 2)
3. **pharmaceutical-compliance** -- If touching clinical data, understand GxP and FDA requirements
4. **legal-document-automation** -- Automate BAA (Business Associate Agreement) generation
5. **quality-control** -- Establish QMS for software validation and change control

### Workflow 2: Optimizing a Distribution Network

1. **logistics-optimization** -- Redesign transportation routes and network topology
2. **warehouse-operations** -- Optimize pick, pack, and ship workflows in each facility
3. **import-export** -- Ensure customs compliance for cross-border shipments
4. **production-planning** -- Align production schedules with demand forecasts
5. **retail-analytics** -- Analyze sell-through data to inform demand planning

### Workflow 3: Entering a Regulated Manufacturing Market

1. **automotive-manufacturing** -- Understand IATF 16949 requirements and production systems
2. **quality-control** -- Establish inspection procedures and statistical process control
3. **production-planning** -- Design MRP and shop floor scheduling systems
4. **safety-management** -- Implement OSHA-compliant safety programs
5. **energy-management** -- Set up energy auditing and ISO 50001 compliance for the facility
