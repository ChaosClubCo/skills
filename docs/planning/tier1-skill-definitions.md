## HOW TO USE THIS PLAYBOOK

### Before You Start
1. Open this file (TIER1_SKILL_DEFINITIONS.md)
2. Open ENHANCEMENT_STATUS.json in editor (side-by-side)
3. Open PowerShell
4. Navigate to: `D:\02_Development\Skills`

### For Each Batch
1. Pick batch (1-5) from the table above
2. Run enhancement script for that batch
3. Manually review each skill in the batch (15 min per skill)
4. Update this file: change "PENDING" → "ENHANCED"
5. Update ENHANCEMENT_STATUS.json: mark batch as COMPLETE
6. Move to next batch OR stop and resume later

### If You Stop Mid-Batch
Example: You're in Batch 2, just finished skill 20 (github-actions-templates)
1. Update this file: change skill 20 from "PENDING" → "ENHANCED"
2. Update ENHANCEMENT_STATUS.json:
   ```json
   "A3_batch_2": {
     "status": "IN_PROGRESS",
     "last_skill_completed": 20,
     "next_skill": "api-development"
   }
   ```
3. Save both files
4. When you resume: Start at skill 21 (api-development)

### If You Stop Between Batches
Example: Batch 2 complete, about to start Batch 3
1. Ensure all Batch 2 skills marked "ENHANCED" in this file
2. Update ENHANCEMENT_STATUS.json:
   ```json
   "A3_batch_2": { "status": "COMPLETE" },
   "A3_batch_3": { "status": "PENDING" }
   ```
3. When you resume: Start Batch 3, skill 31 (api-design-principles)
