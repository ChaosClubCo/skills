# Meeting Facilitation

## Overview

Meeting facilitation is the practice of designing, leading, and documenting meetings to maximize productivity and achieve objectives. For SMBs, effective meetings are critical because limited resources mean every hour counts - poorly run meetings drain time and energy that could be spent on value-creating work.

This skill provides frameworks for planning, running, and following up on meetings of all types. The focus is on practical techniques that transform meetings from necessary evils into productive collaboration sessions.

### Why Meeting Facilitation Matters for SMBs

- **Time Recovery**: Professionals spend 23+ hours/week in meetings; efficiency improvements have huge impact
- **Decision Quality**: Structured facilitation leads to better decisions
- **Team Alignment**: Effective meetings create shared understanding
- **Accountability**: Clear action items drive results
- **Culture**: Meeting quality reflects organizational health

## When to Use

### Primary Triggers
- Planning recurring team meetings or stand-ups
- Preparing for important client or stakeholder meetings
- Establishing meeting rhythms and cadences
- Improving meeting effectiveness and documentation
- Facilitating workshops or decision-making sessions

### Specific Use Cases
- "How do I run more effective team meetings?"
- "What should be in our meeting agenda template?"
- "Help me document meeting decisions and action items"
- "What meeting cadence should we establish?"
- "How do I facilitate virtual meetings better?"
- "We need a RACI matrix for this project"
- "How do I handle difficult meeting dynamics?"

## Core Processes

### 1. Meeting Planning Framework

#### Meeting Purpose Assessment

```
Before Scheduling Any Meeting, Answer:
1. What is the specific objective?
2. What type of meeting is this?
   □ Information sharing (could be async?)
   □ Discussion/brainstorming
   □ Decision making
   □ Problem solving
   □ Relationship building
3. Who absolutely must attend?
4. What preparation is required?
5. What's the expected outcome?

If Unclear on Any Answer → Meeting May Not Be Needed

Alternatives to Meetings:
- Email for simple information sharing
- Slack/Teams for quick questions
- Recorded video for demonstrations
- Shared documents for collaboration
- Async surveys for gathering input
```

#### Action Item Best Practices

```
SMART Action Items:
- Specific: Clear description of what needs to be done
- Measurable: Know when it's complete
- Assignable: Single owner (not "the team")
- Realistic: Achievable in timeframe
- Time-bound: Specific due date

Good Examples:
✓ "Draft proposal outline for Client X - John - Dec 15"
✓ "Schedule kickoff meeting with vendor - Sarah - by EOD Friday"
✓ "Review and approve budget spreadsheet - Mike - before Monday standup"

Bad Examples:
✗ "Work on proposal" (no owner, deadline, or specificity)
✗ "Team to discuss timeline" (no single owner)
✗ "Finish project soon" (no specific date)
```

### 7. RACI Matrix

#### RACI Framework

```
R = Responsible: Does the work
A = Accountable: Ultimately answerable (only one per task)
C = Consulted: Provides input before decision
I = Informed: Notified after decision

Rules:
- Every task must have exactly one A
- Every task should have at least one R
- R and A can be the same person
- Minimize C's to avoid bottlenecks
- I's receive communication, not requests
```

```markdown

| [Task 2] | I | A | R | C | - |
```

### 8. Virtual Meeting Best Practices

#### Virtual Meeting Checklist

```markdown
## PRE-MEETING
- [ ] Test technology (video, audio, screen share)
- [ ] Send agenda and materials 24+ hours ahead
- [ ] Include clear video link in invite
- [ ] Prepare backup communication method
- [ ] Set up virtual whiteboard if needed
- [ ] Plan for engagement activities

## DURING MEETING
- [ ] Start 2-3 min early for tech issues
- [ ] Establish ground rules (mute, video, chat)
- [ ] Do quick audio/video check
- [ ] Use participant names when speaking
- [ ] Share screen for visual focus
- [ ] Check in with remote participants regularly
- [ ] Use chat for questions/parking lot
- [ ] Take breaks every 45-60 min for long meetings
- [ ] Summarize decisions verbally

## POST-MEETING
- [ ] Send notes within 24 hours
- [ ] Include recording link (if recorded)
- [ ] Highlight action items clearly
- [ ] Confirm next steps
```

#### Virtual Facilitation Techniques

```
Engagement Techniques:
1. Round Robins: Go around to each person
2. Chat Storms: Everyone types response simultaneously
3. Polls: Quick voting on options
4. Breakout Rooms: Small group discussions
5. Whiteboard: Visual collaboration
6. Reaction Buttons: Quick feedback
7. Hand Raise: Queue for speaking

Attention Maintenance:
- Keep segments under 10 minutes
- Change activities frequently
- Use visuals, not just talking
- Call on people by name
- Use "popcorn" style (person picks next speaker)
- Build in movement/stretch breaks
```

## Common Pitfalls

### 1. No Clear Purpose
**Problem**: Meetings without defined objectives waste time
**Prevention**:
- Require objective statement on all invites
- "No agenda, no attenda" policy
- Cancel meetings without clear purpose

### 2. Wrong Attendees
**Problem**: Too many or wrong people in meetings
**Prevention**:
- Required vs optional attendee distinction
- Minimum viable audience principle
- Allow people to decline or leave

### 3. Poor Time Management
**Problem**: Meetings run over, start late
**Prevention**:
- Time-boxed agenda items
- Visible timer during meeting
- Hard stops honored
- Parking lot for tangents

### 4. Lack of Follow-Through
**Problem**: Decisions and action items not tracked
**Prevention**:
- Document decisions in meeting
- Assign owners and dates to all actions
- Review previous actions at start
- Track completion rates

### 5. Meeting Overload
**Problem**: Calendar full of meetings, no time for work
**Prevention**:
- Meeting-free blocks/days
- Audit meeting necessity regularly
- Default to async unless sync needed
- Shorter default durations (25/50 vs 30/60)

### 6. Virtual Disengagement
**Problem**: Remote participants check out
**Prevention**:
- Camera-on expectations
- Frequent engagement techniques
- Shorter meeting segments
- Direct address by name
