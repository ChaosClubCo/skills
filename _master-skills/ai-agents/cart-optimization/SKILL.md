## Overview

Cart Optimization focuses on maximizing conversion from cart to purchase while increasing average order value. This skill covers checkout flow design, abandonment recovery, and strategic upselling to capture maximum value from purchase intent.

### Cart Optimization Fundamentals

1. **Checkout Flow** - Streamlined purchase process
2. **Abandonment Recovery** - Recapturing lost carts
3. **Cart Upsells** - Increasing order value
4. **Trust Building** - Reducing purchase anxiety
5. **Payment Optimization** - Removing payment friction

## Common Pitfalls

### Checkout Errors

1. **Required Account Creation**
   - Forcing registration
   - Lost conversions
   - Friction increase

2. **Surprise Costs**
   - Hidden shipping
   - Late tax display
   - Fee disclosure

3. **Form Friction**
   - Too many fields
   - Poor validation
   - Mobile issues

### Recovery Mistakes

1. **Poor Timing**
   - Too fast (aggressive)
   - Too slow (lost interest)
   - Wrong frequency

2. **Weak Incentives**
   - Discount too small
   - No clear value
   - Generic messaging

3. **Missing Segmentation**
   - Same message for all
   - Ignoring cart value
   - No personalization

### Upsell Errors

1. **Irrelevant Offers**
   - Unrelated products
   - Wrong price point
   - No clear value

2. **Overwhelming Display**
   - Too many options
   - Cluttered cart
   - Confusion

3. **Wrong Placement**
   - Blocking checkout
   - Too late in flow
   - Too aggressive

## Best Practices

### Checkout Design

1. **Simplify** - Remove unnecessary steps
2. **Transparency** - Show all costs early
3. **Progress** - Indicate steps remaining
4. **Mobile-First** - Optimize for mobile

### Recovery

1. **Quick Response** - First email within 1 hour
2. **Personalization** - Use cart contents
3. **Incentive Ladder** - Increase over time
4. **Multi-Channel** - Email + SMS + retargeting

### Upselling

1. **Relevance** - Related products only
2. **Value Clear** - Show benefit
3. **Limited Options** - 2-3 suggestions max
4. **Right Timing** - Don't block purchase

### Testing

1. **A/B Test Everything** - Continuous testing
2. **Mobile Focus** - Test on devices
3. **Full Funnel** - Test complete flow
4. **Statistical Significance** - Valid sample sizes
