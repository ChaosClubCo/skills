# Presentation Design

## Overview

Presentation design transforms information and narratives into compelling visual stories that engage audiences and drive action. This skill encompasses strategic slide design, visual hierarchy, data visualization, and the creation of presentation systems that enable teams to communicate effectively.

Effective presentation design balances aesthetics with clarity, ensuring that visual elements enhance rather than distract from the core message. This skill covers everything from high-stakes pitch decks to scalable template systems.

### Core Competencies

- Visual storytelling and narrative structure
- Slide composition and layout
- Data visualization for presentations
- Typography and readability
- Image selection and treatment
- Animation and transition design
- Template system creation
- Speaker support and notes

## When to Use

### Primary Applications

1. **Pitch Decks**
   - Investor presentations
   - Sales pitches
   - Partnership proposals
   - Funding rounds

2. **Corporate Presentations**
   - Board meetings
   - Quarterly reviews
   - Strategy presentations
   - All-hands meetings

3. **Conference and Events**
   - Keynote presentations
   - Workshop slides
   - Training materials
   - Speaker decks

4. **Template Systems**
   - Company-wide templates
   - Department-specific designs
   - Self-service slide libraries
   - Master slide systems

### Trigger Conditions

- High-stakes presentation needed
- Current slides underperforming
- Brand consistency required across decks
- Complex information needs visualization
- Professional polish required
- Template system scaling needed

## Common Pitfalls

### Design Mistakes

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| Too much text | Audience reads, not listens | One idea per slide |
| Inconsistent design | Unprofessional appearance | Use master slides |
| Poor contrast | Unreadable content | Test on projector |
| Clip art/stock | Dated appearance | Custom or quality images |

### Content Issues

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| No clear story | Confused audience | Outline narrative first |
| Data overload | Overwhelming charts | Simplify ruthlessly |
| Missing CTA | No action taken | Clear ask every time |
| Too long | Lost attention | Respect time limits |

### Technical Problems

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| Font issues | Text displays wrong | Embed or use safe fonts |
| Broken links | Dead-end content | Test all links |
| Video fails | Awkward pause | Embed or backup plan |
| Wrong format | Incompatibility | Confirm platform |

## Quality Checklist

### Content

- [ ] Clear narrative arc
- [ ] One idea per slide
- [ ] Headlines tell the story
- [ ] Data visualized effectively
- [ ] Sources cited where needed
- [ ] Spelling and grammar verified

### Design

- [ ] Brand guidelines followed
- [ ] Consistent spacing and alignment
- [ ] Typography readable at distance
- [ ] Color contrast adequate
- [ ] Images high quality
- [ ] Visual hierarchy clear

### Technical

- [ ] Animations purposeful and consistent
- [ ] Links and embeds working
- [ ] Fonts embedded or safe
- [ ] File size optimized
- [ ] Backup file created
- [ ] All formats exported

### Delivery Support

- [ ] Speaker notes complete
- [ ] Timing notes included
- [ ] Transition cues marked
- [ ] Handout version created
- [ ] Presenter trained
