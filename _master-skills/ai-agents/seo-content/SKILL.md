# SEO Content

## Overview

SEO content creation develops search-optimized content that attracts organic traffic while delivering genuine value to readers. This skill combines keyword research, content strategy, and quality writing to create content that ranks well in search engines and converts visitors into customers.

Effective SEO content serves two audiences simultaneously: search engines that determine rankings and humans who read and act on the content. This skill addresses both technical optimization and reader engagement.

### Core Competencies

- Keyword research and selection
- Search intent analysis
- Content structure for SEO
- On-page optimization
- Pillar/cluster strategy
- Meta content creation
- Content refresh and optimization
- Performance tracking

## When to Use

### Primary Applications

1. **Pillar Content**
   - Comprehensive topic guides
   - Ultimate guides
   - Resource centers
   - Evergreen content

2. **Blog Content**
   - Keyword-targeted posts
   - How-to guides
   - Listicles
   - Industry analysis

3. **Landing Pages**
   - Product/service pages
   - Category pages
   - Location pages
   - Feature pages

4. **Content Optimization**
   - Existing content refresh
   - Ranking improvement
   - CTR optimization
   - Conversion optimization

### Trigger Conditions

- Organic traffic growth goal
- Content gap identified
- Keyword opportunity found
- Existing content underperforming
- New product/service launch
- Competitive content needed

## Common Pitfalls

### Content Mistakes

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| Keyword stuffing | Poor experience, penalties | Natural integration |
| Thin content | No value, no ranking | Comprehensive coverage |
| Wrong intent | Mismatch with SERP | Analyze intent first |
| Duplicate content | Cannibalization | Unique angles |

### Technical Issues

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| Slow page load | Poor rankings | Optimize speed |
| Poor mobile | Lost traffic | Mobile-first |
| Missing meta | Low CTR | Complete metadata |
| No schema | Missing features | Implement schema |

### Strategic Problems

| Pitfall | Problem | Prevention |
|---------|---------|------------|
| High difficulty only | No rankings | Balance difficulty |
| Ignoring intent | Wrong content | Intent analysis |
| Never updating | Decay | Refresh schedule |
| No measurement | Unknown ROI | Track everything |

### Pre-Writing

- [ ] Keyword research complete

- [ ] Content brief approved
- [ ] Competitive gaps identified

### Writing

- [ ] Primary keyword in H1
- [ ] Keywords in first 100 words
- [ ] Subheadings include keywords
- [ ] Natural keyword distribution
- [ ] Comprehensive coverage
- [ ] Unique insights included
- [ ] Scannable formatting

### On-Page

- [ ] URL optimized
- [ ] Meta title optimized (50-60 chars)
- [ ] Meta description compelling (150-160 chars)
- [ ] Internal links (3-5+)
- [ ] External authoritative links
- [ ] Images with alt text
- [ ] Schema markup applied

### Technical

- [ ] Mobile responsive
- [ ] Fast load speed
- [ ] Proper heading hierarchy
- [ ] Canonical set
- [ ] Index status verified
