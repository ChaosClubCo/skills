# Canvas Design Skill

This skill guides creation of visual art through a two-step process: **Design Philosophy Creation** followed by **Canvas Expression**.

## Key Principles

**Philosophy Development**: Craft an aesthetic movement (4-6 paragraphs) that emphasizes visual communication over text. The philosophy should articulate how ideas manifest through space and form, color and material, scale and rhythm, composition and balance.

**Craftsmanship Emphasis**: Work must appear meticulously labored. Instructions stress this repeatedly—phrases like "painstaking attention" and "master-level execution" signal that output should evoke countless hours of expert refinement.

**Minimal Text Integration**: "Text as visual accent" drives the approach. Information lives in design elements, not explanation. Typography serves composition, appearing sparingly and whisper-quiet.

**Conceptual Depth**: The skill introduces "deducing the subtle reference"—embedding niche concepts invisibly into form, like a jazz musician quoting another song. Only those who know will catch it.

## Canvas Execution Standards

Output formats: **.pdf, .png, or .md files only**

The final piece should feel like museum-quality work—as if it took countless hours and could prove expertise and rank as undeniably impressive. Composition demands flawless spacing, zero overlaps, and intentional color palettes.

## Critical Guardrail

Create original visual designs, never copying existing artists' work to avoid copyright violations.

## Process

1. **Design Philosophy First**: Write 4-6 paragraphs describing the aesthetic movement, visual language, and conceptual underpinning before creating anything
2. **Canvas Expression**: Translate the philosophy into a concrete visual piece using the output format most appropriate to the brief
3. **Quality Check**: Verify spacing, color harmony, and that the piece communicates the intended concept without relying on explanatory text
